Documentation for Experiment¶

The Experiment class organizes the data from Selection objects into experimental conditions and replicate analyses.

Experiment¶

class experiment.Experiment(config)¶

Class for a coordinating multiple Selection objects. The Selection objects are assigned as replicates of experimental conditions. Creating an Experiment requires a valid config object, usually from a .json configuration file.

enable_logging(log)¶

Turns on log output for this object. Messages will be sent to the open file handle log.

Note

One log file is usually shared by all objects in the analysis.
